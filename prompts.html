Feature request Apr 23, 2026
"I want to do a website that can track the on sale items at target stores so that I know the discount as soon as it comes out. also add a dedicated page on the website that record all prompts we did here and give a short explanation on what the prompts did."
The core feature request that spawned this entire project. Claude read the existing CSIT 212 project files to understand the style, then planned a two-page site: a Target Sale Tracker and a Prompt Log. The tracker features a Target-red colour scheme, category tabs (Electronics, Clothing, Home, Food, Toys, Beauty, Sports, Pet), a search bar, sort controls, per-item watchlist saved to localStorage, auto-refresh every 30 minutes with a live progress bar, a toast notification system, and a demo catalogue of 35+ realistic sale items. A LIVE mode flag is included so a Node.js proxy (see target_proxy.js) can feed real Target data once set up.
