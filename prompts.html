The core feature request that spawned this entire project. Bug fix Apr 23, 2026
"this link says: Cannot GET /"
The deployed site returned a 404 because the vercel.json routed every request — including the homepage — through the Express function, but express.static doesn't work inside Vercel's serverless runtime. Claude fixed this by splitting the routing: HTML files are now served directly by Vercel's static layer (@vercel/static), while only /api/* routes go to the Node.js function. Redeployed and verified HTTP 200.
